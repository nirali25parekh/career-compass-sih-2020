# sih-app-2020
An app that takes an EQ test and suggests career options based on the personality type of the user

Used:
Machine Learning    
React Native      
MongoDB Atlas     

To run:
```
cd <folder-name>
npm install
cd sihcareerapi-master
flask run
cd ..
adb reverse tcp:5000 tcp:5000
react-native run-android

```
